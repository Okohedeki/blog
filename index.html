Airlock ↗

Run any AI agent — LangGraph, smolagents, CrewAI, OpenAI Agents, Claude — as a controlled, self-hosted HTTP service. You get an OpenAI-compatible URL and control over every step, tool call, and dollar from inside the loop. Workers run as Docker containers by default.

● most active agent runtime · + showcase & landing site
VaultOP ↗

Local-first, MIT-licensed media pipeline for content creators. The vault is the spine — an analyzed, encrypted, content-addressed library — and editing, compilations, and blur are operations on it. On-device AI (Whisper transcription, CLIP tags, object-detection blur). Nothing leaves the machine.

electron on-device AI v0.1.6
markov-engine ↗

The storage-agnostic knowledge engine behind Markov. Give it a URL; it extracts and transcribes content, pulls a summary plus entities and relationships with an LLM, embeds it, and clusters it into the most similar existing Chain — then grows that Chain by searching the web. Pluggable LLM/embedding backends, runs with no keys.

python pgvector / LLM
Function War RL ↗

A reinforcement-learning setup for Function War, a 2-player turn-based artillery game where each shot is a math curve y = f(x). V1 is the simplest end-to-end pipeline: a headless game core, one Gymnasium env, and a compact PPO trainer on PufferLib. Beat the baseline first, add sophistication later.

RL / PPO gymnasium
