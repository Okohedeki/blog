airlock ↗

agent control plane

Run any AI agent — LangGraph, smolagents, CrewAI, OpenAI Agents, Claude — as a controlled, self-hosted HTTP service. Today: a Python worker packaged as a Docker image owns the agent loop and serves an OpenAI-compatible /v1/chat/completions endpoint with SSE step-streaming, run-control routes, a live /console, and /metrics. 02

VaultOP ↗

local-first media studio

MIT-licensed media pipeline for creators where nothing leaves the machine. Today (v0.2.2): drag-drop footage is content-hashed, AES-256-GCM encrypted (key in the OS keychain), transcoded, and scene-split into a searchable library. One click builds vertical teasers and cross-library compilations; a mandatory human blur gate guards export. On-device AI runs locally — Whisper transcription, CLIP tags, object-detection blur prefill — via transformers.js. Signed & notarized macOS + Windows builds; drivable headless via CLI and MCP.

v0.2.2 Electron React 19 TypeScript SQLite ffmpeg transformers.js
03

markov-engine ↗

self-growing knowledge engine

A storage-agnostic engine that turns saved links — articles, PDFs, YouTube/TikTok/Reddit, audio — into living Chains of knowledge. Today: give it a URL and it extracts & transcribes content, pulls a summary plus entities/relationships with an LLM, embeds it, and clusters it into the nearest Chain — then grows that Chain by searching the web and synthesizes articles across its sources. All persistence sits behind one Store interface (SQLite ships default); LLM and embedding backends are pluggable and run with no keys.

open-source engine Python LLM / RAG embeddings pgvector / SQLite
04

Function War — RL ↗

reinforcement learning

An RL setup for a 2-player turn-based artillery game where each shot is a math curve y = f(x). Today (V1): a deterministic, tested headless game core (19 tests), a single minimal Gymnasium env, a tiny continuous actor-critic, and a compact PPO trainer over PufferLib's vectorized envs — with eval win-rate against scripted and random opponents. Deliberately stripped down: beat the baseline first, add sophistication later.

V1 training PyTorch PPO Gymnasium PufferLib
